# 📸 E-commerce Product Card (UI Design)

> Um componente de exibição de produto moderno e responsivo, desenvolvido com HTML5 e CSS3, apresentando alta fidelidade visual e foco na experiência de compra (UX).

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi desenvolver um **Cartão de Produto** (Product Card) robusto para e-commerces, focado na apresentação clara de detalhes técnicos, preços dinâmicos e chamadas para ação (CTAs) eficientes.

O layout separa de forma inteligente a área de descrição e especificações da área visual e de marca, garantindo que o usuário tenha todas as informações necessárias para a tomada de decisão em um único componente.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura Modular:** Divisão do componente em sub-blocos lógicos (`box one` para detalhes e `box two` para mídia), facilitando a reutilização do código em diferentes contextos de layout.
* **Sistema de Avaliação (Rating):** Implementação de um feedback visual de satisfação utilizando a biblioteca **Font Awesome**, permitindo uma comunicação imediata de prova social.
* **Gestão de Preços e Descontos:** Estrutura dedicada para exibição de preços originais e com desconto, essencial para estratégias de conversão em vendas.
* **Interface Orientada a Conversão:** Inclusão de botões de ação distintos ("Add no carrinho" e "Login para mais"), demonstrando compreensão sobre fluxos de usuário e CTAs.
* **Design de Atributos:** Destaque para selos de confiança, como "Frete grátis", e identificação clara de marca (`brand`) e modelo (`name`).
* **Organização Profissional:** Código estruturado de forma limpa, com separação total entre a estrutura semântica (`index.html`) e a identidade visual (`css/style.css`).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação de componentes de interface.
* **CSS3:** Estilização de layout e gerenciamento de core.
* **Font Awesome (v5.15.2):** Iconografia vetorial para suporte visual.
